# Note Craft Mobile 📱

Bagian ini berisi kode sumber untuk aplikasi mobile **Note Craft** yang dikembangkan menggunakan Flutter.

Untuk informasi lengkap mengenai proyek, visi, tim, dan infrastruktur backend, silakan merujuk ke [README Utama di Root Project](../README.md).

---

## 🚀 Memulai (Mobile)

1. Pastikan Flutter SDK sudah terinstal.
2. Jalankan `flutter pub get` untuk menginstal dependensi.
3. Jalankan `flutter run` untuk memulai aplikasi dalam mode debug.

### Membangun Rilis (APK)
Gunakan perintah berikut untuk membangun APK yang dioptimalkan berdasarkan arsitektur perangkat :

```bash
flutter build apk --release --split-per-abi
```
