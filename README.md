# NoteCraft Starter Project

This project contains two main parts:
1. `mobile/`: Flutter mobile application.
2. `backend/`: Elysia + Bun backend.

## Getting Started
k
### Backend
1. Go to the `backend` directory.
2. Run `bun install` (already done during initialization).
3. Start the server with `bun dev` (or `bun run index.ts`).
4. API documentation will be available at `http://localhost:3000/swagger`.

### Mobile
1. Go to the `mobile` directory.
2. Run `flutter pub get`.
3. Run the app with `flutter run`.

## Technology Stack
- **Mobile**: Flutter 3.x
- **Backend**: Bun + ElysiaJS
- **API Documentation**: Swagger/OpenAPI
